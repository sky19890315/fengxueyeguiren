== Description ==

With this plugin you can create Back To Top custom for the your website Wordpress.
It's very simple and easy to use. And alway FREE. Give me a cup of coffee if you can.

### Features

* Create custom the scroll option.
* Create custom scroll speed.

GP Back To Top plugin is created by <a href="http://gianguru.net" target="_blank">Giang Peter</a>

### Source code?

You can get GP Back To Top source code on our <a href="http://github.com/giangmd/gp-back-to-top" target="_blank">GP Back To Top</a>


== Installation ==

1. Upload gp-back-to-top.zip to plugins via WordPress admin panel or upload unzipped folder to your wp-content/plugins/ folder
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Settings -> GP Back To Top to setting
